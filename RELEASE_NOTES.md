# TG 内容工具箱 v0.3.0（Windows 便携版）

这是面向普通 Windows 用户的单文件便携版。下载本页附件中的：

```text
TG_Content_Toolbox_v0.3.0.exe
```

双击即可运行，不需要安装 Node.js、Git 或其他开发环境，也不需要执行 BAT、VBS 或命令行脚本。

## 第一次使用

1. 下载 EXE 到任意可写文件夹。
2. 双击启动。便携版约 400 MB，第一次启动需要解压并接受 Windows 安全扫描，部分电脑可能等待约一分钟。
3. 本软件目前没有数字签名；如果 Windows SmartScreen 显示“未知发布者”，请确认文件来自本仓库 Release，再选择“更多信息 -> 仍要运行”。
4. 软件会为每位用户创建独立的本地数据库，不会附带发布者的番号、Telegram 消息、Token、Cookie 或账号数据。
5. 打开“工具首页”，选择需要使用的工具。

## 四个独立工具

### 推特博主

从 Telegram 消息、粘贴文字或多个导出文件中提取博主名，并分别输出：

- 一行一个博主名；
- 一行一个 `https://x.com/用户名` 主页链接。

### Bad.news 帖子

提取规范的 `https://bad.news/t/数字` 帖子链接，自动去除 `/app`、下载页、查询参数和官网其他无关链接。

### MissAV

- 从 Telegram、TXT、HTML、JSON、CSV 等内容中过滤和规范番号；
- 查询 MissAV 详情页并抓取女优 Tag、类型 Tag；
- 维护永久番号库、批次、标签、异常和导出记录；
- 支持网络错误重跑、按标签导出和预计剩余时间；
- Raindrop 同步前必须人工预览和确认；
- 根据批次创建前的女优底库，把结果自动预分流到根目录 `missav1` 或 `missav2`。

### 123AV

- 与 MissAV 使用完全独立的输入、批次、速度和结果页面；
- 查询必须由用户手动开始，不会因为运行 MissAV 而自动启动；
- 查询命中后可选择 Chrome 扩展、APP 内执行器或仅导出；
- 同一网站的账号收藏固定单路执行，避免并发触发频率限制；
- 已收藏项目会跳过，状态不明确时进入复查，不会盲目重复点击。

## Telegram 来源

最多选择 5 个指定群组，可使用：

- Telegram Bot：推荐用于以后新增消息；
- Telegram Desktop HTML/JSON 导出：推荐用于过去历史；
- Telegram 个人账号 API：可选高级入口。

每个工具可绑定自己的群组，同一个群也可以同时绑定多个工具。Telegram Token、账号会话和 API 凭据使用 Windows 安全存储保护，不写入 SQLite 或运行日志。

## 数据库与隐私

- 番号库使用本机 SQLite，可支持十万条以上记录；
- 支持搜索、分页、单条编辑、Windows 文件管理器式多选、批量修改和删除；
- 支持备份、恢复、数据体检、全库归零和数据库位置迁移；
- 更换新版 EXE 不会自动覆盖已有数据库；
- 正式操作前建议在“番号库 -> 维护 -> 备份恢复”创建备份；
- 软件不会随安装包上传或共享用户的数据库、Telegram 正文、密码、Token、Cookie 或浏览器存储。

## 外部账号说明

- Telegram Bot 需要用户自己的 Bot Token；
- Telegram 个人账号 API 需要用户自己的 `api_id` 和 `api_hash`；
- Raindrop 直接同步需要用户自己的 Access Token；
- 123AV 收藏需要用户在所选的 Chrome 或 APP 账号窗口中自行登录；
- 登录、验证码、CAPTCHA 和站点访问限制必须由用户手动处理。
